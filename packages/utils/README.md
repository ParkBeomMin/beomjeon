# 🧰 @beomjeon/utils

범전 위키에서 사용하는 재사용 가능한 유틸 함수 모음입니다.

## ✨ Features

- `isEmpty` - null, 빈 문자열, 빈 배열, 빈 객체 체크

## 📦 Install

```bash
# pnpm
pnpm add @beomjeon/utils

# npm
npm install @beomjeon/utils

# yarn
yarn add @beomjeon/utils

# bun
bun add @beomjeon/utils

```

## 📚 Usage

```typescript
import { isEmpty } from "@beomjeon/utils";

// 빈 값 체크
if (isEmpty(value)) {
    console.log("값이 없습니다.");
}
```
